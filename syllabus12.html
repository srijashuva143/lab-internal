Looping Statements in JavaScript

In JavaScript, there are several types of loops to iterate over arrays, objects, or to execute code repeatedly. The most common looping statements are for loop, while loop, and do-while loop.

1. For Loop:

for (initialization; condition; increment/decrement) {
    // code block to be executed
}

2. While Loop:

while (condition) {
    // code block to be executed
}

3. Do-While Loop:

do {
    // code block to be executed
} while (condition);

Explanation:

  • initialization: An expression (including assignment expressions) or variable declaration. Typically used to initialize a counter variable.
  • condition: An expression evaluated before each iteration. If this expression evaluates to true, the loop continues. If false, the loop ends.
  • increment/decrement: An expression to be evaluated after each iteration of the loop. Typically used to increment or decrement the counter variable.

Example:

let fruits = ['Apple', 'Banana', 'Orange'];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

let i = 0;
while (i < fruits.length) {
    console.log(fruits[i]);
    i++;
}

let j = 0;
do {
    console.log(fruits[j]);
    j++;
} while (j < fruits.length);

This code snippet will output:

Apple
Banana
Orange
Apple
Banana
Orange
