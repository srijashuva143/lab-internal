Switch Case Statement in JavaScript

The switch case statement in JavaScript is used to perform different actions based on different conditions. It is a type of conditional statement that evaluates an expression, matches the expression value to a case label, and executes the code block associated with that case.

Here's the basic syntax of a switch case statement:

switch(expression) {
    case label1:
        // code block for label1
        break;
    case label2:
        // code block for label2
        break;
    default:
        // default code block
}

Explanation of the parts:

  • switch(expression): The expression is evaluated once. The value of the expression is compared with the values of each case label.
  • case label: A case label is used to match the value of the expression. If the value matches, the code block associated with that case is executed.
  • break: The break statement is used to terminate the switch case statement. If omitted, the code execution will "fall through" to the next case.
  • default: The default case is optional. It is executed if no case matches the expression value.

Example:

let fruit = 'Apple';
switch(fruit) {
    case 'Apple':
        console.log('It is an apple.');
        break;
    case 'Banana':
        console.log('It is a banana.');
        break;
    default:
        console.log('Unknown fruit.');
}

This code snippet will output: It is an apple.
